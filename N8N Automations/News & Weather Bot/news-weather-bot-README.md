# 🌤️ Daily News + Weather Bot

An automated n8n workflow that delivers a **daily morning digest** — top 3 global news stories + live weather report for Bhilwara — straight to Telegram at 9:00 AM.

Built and deployed in **June 2025**.

---

## What It Does

Every morning at 9 AM, the bot automatically:
- Fetches **live weather data** for Bhilwara (temperature, humidity, wind speed, conditions)
- Scrapes the **top 3 global and domestic news headlines** from live sources
- Formats everything into a clean, readable message in **Hindi + English**
- Sends it directly to a **Telegram chat** — zero manual input

---

## Tools & Stack

| Tool | Role |
|------|------|
| n8n | Workflow orchestration + scheduling |
| Weather API | Live weather data for Bhilwara |
| News API / RSS | Global & domestic news scraping |
| Telegram Bot API | Message delivery |
